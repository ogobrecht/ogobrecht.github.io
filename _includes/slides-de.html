HTML-Präsentation in einem neuen Fenster/Tab starten

Es wird ein moderner Browser mit aktiviertem JavaScript benötigt. Tips zur Navigation:

  • Space = Nächste Folie
  • Shift + Space = Vorherige Folie
  • Esc = Übersicht aller Folien
  • ? = Übersicht aller Tastaturkürzel
  • Alle enthaltenen Links werden in einem neuen Fenster/Tab geöffnet

Alternativ kann man sich die Präsentation auch als einfache HTML-Datei auf GitHub anschauen. Dann sieht man alle Anmerkungen, allerdings funktionieren JavaScript Elemente nicht, da einfach der Markdown-Quelltext ohne das Reveal.js Template gerendert wird.
